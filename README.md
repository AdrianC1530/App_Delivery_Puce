# PUCE-SI Delivery App 🚀🍔

**App Delivery Puce** es una aplicación móvil desarrollada con Flutter y Firebase, diseñada exclusivamente para mejorar la experiencia gastronómica y comercial dentro del campus de la Pontificia Universidad Católica del Ecuador Sede Ibarra (PUCE-SI). 

Esta plataforma conecta a los estudiantes con los bares y papelerías de la universidad, permitiéndoles realizar pedidos en tiempo real, evitar largas filas y descubrir nuevos emprendimientos estudiantiles.

## ✨ Características Principales

*   **🛒 Menús Digitales y Carrito:** Explora los productos de todos los bares del campus, agrégalos a tu carrito y realiza tu pedido en segundos.
*   **📡 Seguimiento en Tiempo Real:** Revisa el estado de tu orden en vivo (Pendiente, Preparando, Listo para retirar, En camino y Entregado).
*   **🔐 Roles de Usuario y Panel de Administración:** 
    *   *Estudiantes:* Pueden comprar, revisar su historial de pedidos y apoyar emprendimientos.
    *   *Administrador / Bares:* Tienen acceso a un panel de control exclusivo para encender/apagar locales, gestionar productos, actualizar el estado de los pedidos y aprobar sugerencias.
*   **🚀 Hub de Emprendimientos:** Un espacio dedicado para que los estudiantes publiquen y ofrezcan sus propios productos, sujeto a aprobación por parte de la administración para mantener la calidad.
*   **🔍 Búsqueda Inteligente:** Encuentra exactamente lo que se te antoja escribiendo el nombre del producto o local en el buscador integrado.
*   **📫 Buzón de Sugerencias:** Los estudiantes pueden enviar notas de voz con sugerencias de mejora directamente a la administración.

## 🛠️ Tecnologías Utilizadas

*   **Frontend:** Flutter (Dart) - Arquitectura basada en Providers.
*   **Backend:** Firebase (Authentication & Cloud Firestore).
*   **Diseño:** UI/UX moderno, soporte nativo para Modo Oscuro/Claro y animaciones dinámicas (Glassmorphism).

## 💡 Instalación y Configuración (Para Desarrolladores)

1. Clona este repositorio:
   ```bash
   git clone https://github.com/AdrianC1530/App_Delivery_Puce.git
   ```
2. Instala las dependencias de Flutter:
   ```bash
   flutter pub get
   ```
3. Ejecuta la aplicación en tu emulador o dispositivo físico:
   ```bash
   flutter run
   ```
